## 腾讯精选50题

### 104-给定一个二叉树找出其最大深度
>题目描述:
二叉树的深度为根节点到最远叶子节点的最长路径上的节点数

```
测试用例： 给定二叉树 [3,9,20,null,null,15,7]， 返回最大深度为3
    3
   / \
  9  20
    /  \
   15   7
```

```
解题思路：递归思想出发，每次找左子树和右子树的深度去最大值

```
